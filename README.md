# Airbnb API

REST & GraphQL API of the Airbnb Clone using Django REST Framework and Graphene GraphQL

### API Actions

- [x] List Rooms
- [x] See Room
- [x] Create Room
- [x] Edit Room
- [x] Delete Room
- [x] See Favs
- [x] Add/Remove Room to Favourites
- [x] Edit Profile
- [x] See Profile
- [x] Login (JWT)
- [x] JWT Authentication
- [ ] Search Room
